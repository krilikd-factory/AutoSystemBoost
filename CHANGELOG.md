# ASB V13.2 - User Changelog

## What's New in V13.2

V13.2 is a stability and battery-focused update.

### Better Idle Battery Life
- Reduced unnecessary CPU wakeups
- Improved background task handling
- More stable memory writeback behavior

### Smarter CPU Control
- Proper optimization for OnePlus 15 (6 efficiency + 2 prime cores)
- Background tasks now use the full efficiency cluster
- More responsive performance when needed

### Improved Network Stability
- Cleaner WiFi queue handling
- Stable mobile data behavior
- Better compatibility with carrier settings

### Banking and Payment App Safety
- Fixed settings that could interfere with apps like SuperQi
- Improved Play Integrity compatibility

### Cleaner System Behavior
- Removed non-working kernel tweaks
- Reduced unnecessary logging overhead
- Improved overall stability
