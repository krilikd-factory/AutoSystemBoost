<p align="center">
AutoSystemBoost — это продвинутый <b>модуль оптимизации системы Android</b>, созданный для улучшения общего опыта использования устройства.
Первоначально разработан для <b>OnePlus 15</b>, однако логика оптимизаций полезна для большинства современных флагманских устройств на Qualcomm.
</p>

## ⚡ Что улучшает AutoSystemBoost

AutoSystemBoost оптимизирует несколько ключевых подсистем Android:

| Подсистема | Оптимизация |
|-----|-----|
| CPU Scheduler | Тюнинг WALT планировщика для более плавного распределения задач |
| GPU | Оптимизация поведения GPU и энергопотребления |
| Memory | Улучшенная настройка VM и уменьшение фоновых пробуждений |
| Network | TCP оптимизация для мобильных сетей |
| Audio | Улучшенный динамический диапазон и максимальная громкость |
| Camera | Оптимизация медиапайплайна |
| Wi‑Fi | Оптимизация задержек и сканирования |
| GPS | Более быстрый захват спутников и стабильность |
| System | Снижение лишних системных логов и фоновой активности |

Результат — **более плавная система, меньше нагрев и лучшая автономность**.

## 🔊 Улучшения звука

AutoSystemBoost значительно улучшает **Bluetooth‑аудио и системный звук**.

### 🎧 Что меняется

✔ Более высокая громкость без клиппинга  
✔ Более чистые пики (лучший динамический диапазон)  
✔ Стабильное Bluetooth соединение  
✔ Улучшенная совместимость с качественными кодеками

Особенно заметно с:

- LHDC  
- LDAC  
- AAC  
- флагманскими TWS наушниками

### Пример результата

| Параметр | Сток | С ASB |
|------|------|------|
| Максимальная безопасная громкость | средняя | выше |
| Динамический диапазон | стандарт | улучшен |
| Клиппинг на пиках | возможен | уменьшен |
| Стабильность Bluetooth | обычная | лучше |

Идеально для **OnePlus Buds Pro / Pro 2 / Pro 3** и других качественных TWS.

## 📷 Камера и видео

ASB включает параметры, которые улучшают работу **camera и media pipeline**.

### Улучшения

✔ улучшенная работа кодеков  
✔ стабильность видеозаписи  
✔ улучшенная совместимость HDR / HEVC  
✔ более плавная запись видео с высоким FPS

### Результат

| Параметр | Сток | ASB |
|------|------|------|
| Стабильность видео | нормальная | лучше |
| Эффективность кодеков | стандарт | оптимизирована |
| Запись высокого FPS | нормальная | плавнее |

Эти изменения **не модифицируют приложение камеры**, а оптимизируют **низкоуровневый медиастек Android**.

## 🔋 Автономность

Одна из ключевых целей ASB — **повышение энергоэффективности устройства**.

Модуль уменьшает лишние пробуждения CPU и улучшает поведение системы в простое.

### Ожидаемые улучшения

| Сценарий | Сток | С ASB |
|------|------|------|
| Ночной расход | 4‑6% | **2‑3%** |
| Idle расход | стандарт | **ниже** |
| Повседневное использование | базовое | **+5‑10% автономности** |
